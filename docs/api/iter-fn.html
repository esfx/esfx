@esfx/iter-fn Package

    An iteration and query API for ECMAScript iterables.

    Installation

    npm i @esfx/iter-fn

    Usage

    • TypeScript
    • JavaScript (CommonJS) 
    import { map } from "@esfx/iter-fn";
const x = map([1, 2, 3], x => x * 2);
x; // [2, 4, 6]

    const { map } = require("@esfx/iter-fn");
const x = map([1, 2, 3], x => x * 2);
x; // [2, 4, 6]
