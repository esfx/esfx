@esfx/async-iter-fromsync Package

    An API to convert an ECMAScript Iterable into an AsyncIterable.

    Installation

    npm i @esfx/async-iter-fromsync

    • JavaScript (CommonJS) 
    import { toAsyncIterable } from "@esfx/async-iter-fromsync";

    // ...
