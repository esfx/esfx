@esfx/async-delay Package

    The @esfx/async-delay package provides the delay(msec) function.

    Installation

    npm i @esfx/async-delay

    Usage

    • TypeScript
    • JavaScript (CommonJS) 
    import { delay } from "@esfx/async-delay";

async function doSomeAction() {
    // wait 10 seconds
    await delay(10 * 1000);
}

    const { delay } = require("@esfx/async-delay");
