    source(cancelables)

    Creates a new CancelSource.

    Declaration
    static source(cancelables?: Iterable<Cancelable | DOMAbortSignal | null | undefined>): CancelSource;
    Parameters
    cancelables
    Iterable<Cancelable | DOMAbortSignal | null | undefined>

    An optional iterable of Cancelable objects. If present, the source becomes linked to the provided cancelables and will be canceled when a linked cancelable is canceled.

    Returns
    CancelSource

    Remarks

    Calling CancelToken.source() with cancelables is similar to CancelToken.race(), except you can individually cancel or close the resulting source. This can be better for memory or GC purposes, since when the resulting source is canceled or closed it can be unlinked from the cancelables, removing references from each cancelable to the resulting source which could otherwise prevent garbage collection.
