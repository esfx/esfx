Package type-model

@esfx/type-model

The @esfx/type-model package provides a number of utility types for TypeScript.

Overview

  • Installation
  • Usage
  • API

Installation

npm i @esfx/type-model

Usage

// TypeScript
import { Diff } from "@esfx/type-model";

type A = { x: number, y: string };
type B = { x: number };
type C = Diff<A, B>; // { y: string }
