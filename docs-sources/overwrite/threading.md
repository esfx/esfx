---
uid: threading
---

Provides a number Thread synchronization primitives for use with Workers from the following packages:

- @"threading-autoresetevent"
- @"threading-conditionvariable"
- @"threading-countdown"
- @"threading-lockable"
- @"threading-manualresetevent"
- @"threading-mutex"
- @"threading-semaphore"
- @"threading-sleep"
- @"threading-spinwait"

# Installation

```sh
npm i @esfx/threading
```
