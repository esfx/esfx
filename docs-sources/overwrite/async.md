uid: async
---

Provides a number of asynchronous coordination primitives from the following packages:

- @"async-autoresetevent"
- @"async-barrier"
- @"async-canceltoken"
- @"async-conditionvariable"
- @"async-countdown"
- @"async-deferred"
- @"async-delay"
- @"async-lazy"
- @"async-lockable"
- @"async-manualresetevent"
- @"async-mutex"
- @"async-queue"
- @"async-readerwriterlock"
- @"async-semaphore"
- @"async-stack"
- @"async-waitqueue"

# Installation

```sh
npm i @esfx/async
```
